
# Marketplace Analytics Pro

## Estrutura de Páginas
- `/` - Dashboard (visão geral KPIs)
- `/vendas-diarias` - Vendas Diárias (tabela + gráfico dia a dia)
- `/importacao` - Importação de dados (upload planilhas)
- `/configuracoes` - Configurações (metas, marketplaces)
- `/sellers` - Gestão de Sellers
- `/perfil` - Perfil do usuário
- `/usuarios` - Gestão de Usuários (admin only)
- `/login` - Login
- `/reset-password` - Redefinir senha

## Propósito
Business Intelligence para sellers de marketplaces.
